# ESP Ortam DHT22 + BH1750 Kodu

Bu klasör ortam tarafındaki ESP32 kodunu içerir. Kod DHT22 ile sıcaklık/nem, BH1750 ile ışık şiddeti ölçer ve Flask sunucusundaki `/env` endpointine JSON olarak gönderir.

GitHub sürümünde Wi-Fi adı, Wi-Fi şifresi ve bilgisayar IP adresi gizlenmiştir. Gerçek kullanımda `YOUR_WIFI_SSID`, `YOUR_WIFI_PASSWORD` ve `YOUR_PC_IP` alanları yerel bilgilere göre doldurulmalıdır.
