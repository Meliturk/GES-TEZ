# ESP32 IoT Kodları

Bu klasörde güneş enerji sistemi IoT prototipi için kullanılan ESP32 kodları yer alır.

- `esp_ortam_dht22_bh1750.ino`: DHT22 ve BH1750 sensörleri ile sıcaklık, nem ve ışık şiddeti ölçer.
- `esp_panel_acs712.ino`: ACS712 sensörü ile panel akım ölçümü yapar.

GitHub sürümünde Wi-Fi adı, Wi-Fi şifresi ve yerel IP adresleri gizlenmiştir. Gerçek kullanımda `YOUR_WIFI_SSID`, `YOUR_WIFI_PASSWORD` ve `YOUR_PC_IP` alanları yerel bilgilere göre doldurulmalıdır.
