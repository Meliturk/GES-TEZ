# ESP Panel ACS712 Kodu

Bu klasör panel tarafındaki ESP32 kodunu içerir. Kod ACS712 akım sensörünü ESP32 IO34 pininden okuyup Flask sunucusundaki `/panel` endpointine JSON olarak gönderir.

GitHub sürümünde Wi-Fi adı, Wi-Fi şifresi ve bilgisayar IP adresi gizlenmiştir. Gerçek kullanımda `YOUR_WIFI_SSID`, `YOUR_WIFI_PASSWORD` ve `YOUR_PC_IP` alanları yerel bilgilere göre doldurulmalıdır.
